## Desafio POO
Desafio realizado na plataforma DIO para o bootcamp "Santander 2024 - Backend com Java" com o objetivo de praticar conceitos de Programação orientada a objetos linguagem Java.

O desafio consiste em implementar a estrutura base de classe do modelho do iPhone como foi apresentado pela primeira vez por Steve Jobs em 2007.

**Video de apresentação do iPhone**: https://www.youtube.com/watch?v=9ou608QQRq8&ab_channel=TuchilaRino


**Link do desafio**: https://github.com/digitalinnovationone/trilha-java-basico/tree/main/desafios/poo


## Estrutura

### Funcionalidades a modelar
1. **Reprodutor Musical**
   - Métodos: `tocar()`, `pausar()`, `selecionarMusica(String musica)`
2. **Aparelho Telefônico**
   - Métodos: `ligar(String numero)`, `atender()`, `iniciarCorreioVoz()`
3. **Navegador na Internet**
   - Métodos: `exibirPagina(String url)`, `adicionarNovaAba()`, `atualizarPagina()`
